# Refraction

Refraction will be a game engine that aims to provide high detail realtime graphics at stable and reasonable performance cost without relying on poor excuses like TAA or ML image improvement techniques.
It will also provide a range of ways to implement code such as C++, C#, Lua and visual scripting, however using C++/C# is recommended to use the engine to it's fullest potential. Additionally, there will be a built-in live collaboration system to accelerate team development.
